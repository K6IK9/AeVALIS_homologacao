Instruções de Configuração (.env)

As credenciais de e-mail devem ser configuradas no seu arquivo .env.

Para SMTP:
EMAIL_BACKEND='django.core.mail.backends.smtp.EmailBackend'
EMAIL_HOST='smtp.seuprovedor.com'
EMAIL_PORT=587
EMAIL_USE_TLS=True
EMAIL_HOST_USER='seu-email@provedor.com'
EMAIL_HOST_PASSWORD='sua-senha'
DEFAULT_FROM_EMAIL='seu-email@provedor.com'
Para API SendGrid:
# Remova as variáveis EMAIL_* acima e adicione estas:
SENDGRID_API_KEY='SUA_CHAVE_API_DO_SENDGRID'
DEFAULT_FROM_EMAIL='email_verificado@sendgrid.com'
